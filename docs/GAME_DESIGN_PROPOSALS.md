## 1. 元素反應系統

**優先度：HIGH**

### 機制說明

遊戲中共有五種元素屬性，分別由不同塔台施加於敵人身上：

| 元素     | 符號 | 施加方式               |
|----------|------|------------------------|
| 火焰     | 🔥   | 火焰塔持續燃燒 DoT     |
| 冰霜     | ❄    | 冰塔命中施加「冰凍」狀態 |
| 閃電     | ⚡   | 電塔鏈狀傳導           |
| 毒素     | ☠    | 毒塔持續中毒 DoT       |
| 風元素   | 💨   | 風塔環形氣流           |

當敵人身上同時存在兩種元素狀態時，觸發「元素反應」，產生額外的特殊效果：

### 設計理由

元素反應系統的核心在於「組合發現的驚喜感」。玩家一開始會單純使用強力塔，但隨著了解反應表，會開始刻意安排元素搭配。這種從「被動放塔」到「主動設計 combo」的轉變，讓遊戲深度大幅提升，且每次發現新反應都帶來滿足感。靈感來自原神的元素體系，但針對塔防即時性做了簡化。

## 2. 動態事件卡牌

**優先度：HIGH**

### 機制說明

每隔 3 波敵人，遊戲暫停並彈出「事件卡牌選擇介面」。玩家從 2 張隨機抽取的卡牌中選擇 1 張立即生效。卡牌為該局遊戲帶來不確定性與策略轉折點。

### 卡牌列表

| 卡牌名稱           | 效果                                         | 稀有度 |
|--------------------|----------------------------------------------|--------|
| 流動商隊           | 接下來 1 波：所有塔台建造費用減半             | 普通   |
| 隕石衝擊           | 下一波：敵人數量減少 30%                     | 普通   |
| 大地震裂           | 所有塔退回升級，返還 80% 升級金幣            | 稀有   |
| 詛咒浪潮           | 下一波：敵人速度 ×2，但擊殺金幣 ×2           | 稀有   |
| 義勇軍抵達         | 召喚 3 個友軍單位沿路徑清除一批敵人           | 稀有   |
| 神聖祝福           | 所有塔立即回滿 HP，並獲得 5 秒無敵            | 傳說   |
| 時空裂縫           | 下一波：隨機 2 個塔自動升一級（無需費用）     | 傳說   |

### 設計理由

事件卡牌是打破「塔防節奏單調」的關鍵機制。傳統塔防每波之間只有建塔的選擇，缺乏「戲劇性轉折」。卡牌系統讓每一局都有不同的故事發展，玩家需要根據當前局勢判斷哪張卡更有利。「詛咒浪潮」這類高風險高回報的設計，讓玩家面臨緊張的賭注決策，是製造「遊戲時刻」的好工具。

## 3. 塔台連線增益系統

**優先度：HIGH**

### 機制說明

相同元素屬性的塔台若互相相鄰（上下左右，不含對角），則自動形成「連線網路」。每條連線為兩端塔台各提供 +10% 攻擊速度加成，每座塔最多可建立 4 條連線（上下左右各一），最高獲得 +40% 攻擊速度。

跨元素的特殊組合也有獨特加成：

| 組合               | 特殊效果                         |
|--------------------|----------------------------------|
| 箭塔 + 火焰塔      | 箭塔獲得火焰附魔，增加燃燒傷害   |
| 炮塔 + 冰塔        | 炮塔炮彈命中後有 30% 機率附加減速 |
| 任意塔 + 閃電塔    | 閃電塔的鏈狀傳導多跳 1 次        |

連線增益系統鼓勵玩家「聚集同類塔」，而非隨意分散放置。這創造了一種「集群建造」的規劃樂趣，並且讓先前放置的塔因新增的鄰居而「升值」，帶來持續的策略決策感。此外，同元素集群也與元素反應系統形成協同：聚集的火塔可以快速輸出火焰狀態，再配合一座冰塔觸發大爆炸。

## 4. 英雄單位

**優先度：MEDIUM**

### 機制說明

玩家可在地圖上操控一名英雄角色（WASD 鍵盤移動或滑鼠點擊目標位置）。英雄本身有攻擊力，但主要作用是強化附近的塔台及使用主動技能。英雄有 HP 值，若血量歸零需要 10 秒復活冷卻，復活後不會損失遊戲生命值。

### 技能系統

| 技能名稱     | 快捷鍵 | 效果                                          | 冷卻 |
|--------------|--------|-----------------------------------------------|------|
| 氣場激勵     | Q      | 半徑 150px 內所有塔 +20% 傷害，持續 5 秒      | 20s  |
| 緊急維修     | W      | 修復附近最低血量的塔，恢復 50% 最大 HP        | 15s  |
| 鏈狀閃電     | E      | 釋放閃電擊中最近 5 名敵人，每段傷害遞減 15%   | 12s  |
| 衝刺          | Shift  | 快速衝刺 200px，無敵幀（穿越敵人）            | 8s   |

## 5. 環境高低差

**優先度：MEDIUM**

### 機制說明

地圖格子分為三種地形高度，影響其上方塔台的效果以及通過的敵人：

| 地形類型 | 代號 | 塔台效果          | 敵人效果          |
|----------|------|-------------------|-------------------|
| 高地     | H    | 攻擊範圍 +30%     | 無                |
| 平地     | F    | 標準              | 標準              |
| 谷地     | V    | 攻擊範圍 -10%     | 移動速度 -20%     |

高地格由地圖設計師預設，玩家不可更改地形。部分關卡提供「地形改造卡牌」（搭配事件卡牌系統）可短暫升高一格地形。

### 地形高低示意圖（等距視角）

```
  地圖俯視 + 等距高度示意：

  F = 平地   H = 高地(+1層)   V = 谷地(-1層)

  ┌───┬───┬───┬───┬───┬───┐
  │ F │ H │ H │ F │ F │ V │   Row 0
  ├───┼───┼───┼───┼───┼───┤
  │ V │###│###│###│ F │ F │   Row 1  ### = 路徑（無高差）
  ├───┼───┼───┼───┼───┼───┤
  │ F │ F │ H │ H │ F │ V │   Row 2
  └───┴───┴───┴───┴───┴───┘

  等距立體效果（側視）：
         ___________
        /  H 高地  /|   ← 放這裡的塔射程 +30%
       /___________/ |
      |           |  |
      |  ⬆️ 抬高  | /
      |___________|/

  ___________
  / F 平地  /    ← 標準格，無加成
  __________

      ___________
     /   V 谷地  \   ← 敵人穿越時 -20% 移速
     \___________/    ← 塔放在這裡射程 -10%

  高度視覺編碼（顏色提示）：
  H = 亮色邊框 + 淡藍色背景疊層
  F = 標準顏色，無額外標記
  V = 深色邊框 + 淡棕色加深背景
```

### 設計理由

高低差系統為地圖設計增加了立體感與策略層次，玩家在選擇塔台位置時不只考慮「路徑距離」，還要考慮「地形增益」。高地塔可以覆蓋更廣的路徑但格數有限，谷地雖然削弱塔效能但能讓敵人自然減速，形成各具價值的取捨。此系統也讓「地圖選關」產生更多差異性，每張地圖的高低分佈決定了截然不同的最優策略。

## 6. 陷阱系統

**優先度：MEDIUM**

### 機制說明

陷阱是比塔台便宜的一次性消耗品，只能放置在路徑格上（PATH 格），放置後等待敵人踩踏才觸發。每波開始前可補充購買。

### 陷阱種類

| 陷阱名稱   | 費用 | 觸發條件     | 效果                                           |
|------------|------|--------------|------------------------------------------------|
| 地雷       | 80   | 第 1 名踩踏 | 以踩踏點為中心，120px 範圍內巨量傷害           |
| 荊棘地     | 50   | 連續 10 名  | 每名敵人移速 -40%，並受到持續 3 秒的刺傷 DoT  |
| 油桶       | 40   | 第 1 名踩踏 | 標記範圍內所有敵人，火焰傷害對其翻倍，持續 8s  |
| 假人誘餌   | 60   | 敵人接近時  | 敵人被吸引改道，繞行 3 格額外路徑             |

```
  觸發連鎖示意（油桶 + 荊棘 + 火焰塔）：

  敵人踩油桶 → 標記「易燃」
       ↓
  繼續前進踩荊棘 → 移速 -40%（被減速！）
       ↓
  火焰塔攻擊「易燃」敵人 → 傷害 ×2
       ↓
  ★  COMBO！最佳輸出視窗！ ★
```

### 設計理由

陷阱系統讓路徑格從「死區」變成「策略空間」。傳統塔防玩家傾向忽略路徑本身，有了陷阱後，路徑的每個轉角都成為潛在的伏擊點。低費用一次性的設計使陷阱成為緊急應對的手段，在某波敵人特別強時可以快速補強，而不需花費大量金幣建造永久塔台。

## 7. 敵人進化機制

**優先度：LOW**

### 機制說明

若某個敵人單位成功走到路徑終點（觸發生命值扣除），它不會消失，而是「進化」並從起點再次出現，比上次更強。每次存活累計：

- 第 1 次進化：護甲 +30%、移速 +10%
- 第 2 次進化：獲得能量護盾（額外血條），移速再 +10%
- 第 3 次進化：成為命名 Mini-Boss，獲得一個獨特主動能力

## 8. 時間晶石

**優先度：HIGH**

### 機制說明

「時間晶石」是一種稀有掉落物，擊殺特定種類或精英敵人時有機率掉落（約 15-20% 機率）。收集 3 顆晶石後可以手動激活「時間凍結」能力：

- **激活效果**：持續 5 秒
  - 所有敵人移動速度降低至 20%（幾近靜止）
  - 塔台攻擊速度維持正常（不受影響）
  - 畫面邊框出現藍色時間凍結脈衝特效

- 每場遊戲最多可激活 **2 次**（共需消耗 6 顆晶石）
- 晶石不可轉移至下一關（每關重新計算）
